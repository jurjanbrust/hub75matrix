Device Management APIs

GET /api/status Get device status
GET /api/brightness/increase Increase brightness by 25
GET /api/brightness/decrease Decrease brightness by 25
GET /api/brightness/set?value=X Set brightness (1-255)
GET /api/wifi/reset Reset WiFi credentials
GET /api/restart Restart device
POST /api/gif/upload Upload a GIF file to the /gifs folder

File Management APIs

GET /api/files?path=/folder/ List files and folders in a directory (default: root)
POST /api/files/delete Delete a file or folder.
Body: { "path": "/path/to/file" }
POST /api/files/rename Rename a file or folder.
Body: { "path": "/path/to/file", "newName": "newname" }
POST /api/files/move Move a file or folder.
Body: { "path": "/old/path/file", "newPath": "/new/path/file" }
POST /api/files/create-folder Create a new folder.
Body: { "name": "foldername" }
POST /api/dirs/rename Rename a directory.
Body: { "path": "/old/folder", "newName": "newfolder" }
POST /api/dirs/delete Delete a directory (recursive).
Body: { "path": "/folder/to/delete" }
